# AeroLinks: The Physics of the Flight

Interactive golf ball trajectory simulator demonstrating fluid dynamics principles with an AI-powered Caddy assistant.

<img width="1444" height="715" alt="AeroLinks" src="https://github.com/user-attachments/assets/b58b2603-d968-4a61-b534-dea69ec3495b" />


## Features

- **Realistic Physics Engine**: Simulates ball flight using Magnus effect, drag coefficients, and spin decay
- **Multiple Club Selection**: Driver, 5-Iron, 7-Iron, and Pitching Wedge with accurate parameters
- **Wind Effects**: Headwind and tailwind simulation with adjustable speed
- **AI Caddy**: GPT-powered feedback on your shots with intelligent analysis
- **Animated Visualization**: Watch your ball flight in real-time with Plotly charts
- **IBM Carbon Design**: Clean, professional UI styling

## Installation

### Prerequisites

- Python 3.11+
- [uv](https://docs.astral.sh/uv/) package manager

### Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/timroman234/Aerolinks.git
   cd Aerolinks
   ```

2. Install dependencies:
   ```bash
   uv sync
   ```

3. Configure environment variables:
   ```bash
   cp .env.example .env
   # Edit .env and add your OpenAI API key
   ```

## Usage

Run the Streamlit application:

```bash
uv run streamlit run app.py
```

The app will open in your browser at `http://localhost:8501`.

### Controls

1. **Club Selection**: Choose from Driver, 5-Iron, 7-Iron, or PW
2. **Club Head Speed**: Adjust swing speed (60-120 mph)
3. **Wind Speed**: Set wind magnitude (0-40 mph)
4. **Wind Direction**: Select Headwind or Tailwind
5. Click **Swing** to simulate the shot

## Physics Model

The simulator uses Euler integration to model:

- **Magnus Effect**: Lift generated by ball spin
- **Drag Force**: Air resistance based on velocity squared
- **Spin Decay**: Gradual reduction of backspin over flight time
- **Wind Effects**: Headwind/tailwind impact on drag

| Club   | Launch Angle | Backspin (RPM) | Smash Factor |
|--------|-------------|----------------|--------------|
| Driver | 12°         | 2,500          | 1.48         |
| 5-Iron | 19°         | 5,000          | 1.38         |
| 7-Iron | 27°         | 7,000          | 1.33         |
| PW     | 42°         | 9,000          | 1.23         |

## Environment Variables

| Variable | Description | Required |
|----------|-------------|----------|
| `OPENAI_API_KEY` | OpenAI API key for AI Caddy feature | Optional* |

*The AI Caddy falls back to rule-based responses when the API key is not configured.
